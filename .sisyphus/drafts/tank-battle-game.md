# 草稿: 經典坦克大戰遊戲規劃

## 原始需求
- 環境: Python3, pygame, 使用 uv 進行套件及虛擬環境管理
- 目標: 設計規劃經典的坦克大戰遊戲

## 技術決策 (待確認)
- 遊戲框架: pygame
- 套件管理: uv

## 研究進行中
- [ ] 當前專案結構探索
- [ ] pygame 最佳實踐研究
- [ ] uv 專案管理研究

## 核心需求確認 (使用者已回應)
- **遊戲功能範圍**: 基礎版本（玩家坦克移動/射擊、簡單敵人、基礎地圖）
- **圖片資源來源**: 程序生成（pygame 繪圖 API）
- **預期用途**: 完整作品（可玩的遊戲體驗）

## 範圍邊界
- INCLUDE:
  - 玩家坦克（移動：上下左右；射擊：空格鍵）
  - 簡單敵人（自動移動和射擊）
  - 基礎地圖系統（磚塊、鋼塊）
  - 生命值系統
  - 計分系統
- EXCLUDE:
  - 道具系統（星星、時鐘等加強道具）
  - 關卡進度系統（單一關卡即可）
  - 音效
  - 多人模式（單人遊戲）

## 使用者已確認的參數
- **視窗大小**: 800x600
- **遊戲模式**: 任務模式（消滅所有敵人即可獲勝）
- **地圖佈局**: 隨機生成型（隨機生成障礙物）
- **敵人數量**: 3-5 台同時出現

## 遊戲設計參數
- **地圖類型**: 磚塊（可破壞）、鋼塊（不可破壞）
- **地圖生成**: 隨機散佈障礙物
- **玩家生命值**: 3 顆（經典設計）
- **勝利條件**: 消滅所有敵人
- **失敗條件**: 玩家生命歸零
- **測試策略**: 無自動化測試，使用 Agent-Executed QA 驗證
